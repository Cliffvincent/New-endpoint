Available Endpoints

/ddos/site/spam

Spam a site via link. Example: /ddos/site/spam?link=example.com

/emojimix

Mix two emojis. Example: /emojimix?emoji1=😂&emoji2=❤️

/fbshare

Facebook share bot. Example: /fbshare?link=example.com&token=yourToken&amount=10&speed=fast

/follow

Follow an account. Example: /follow?accountID=12345&accessToken=yourToken

/tempmail/gen

Generate a temporary email.

/tempmail/message

Check temporary email messages. Example: /tempmail/message?email=temp@example.com

/guard

Enable or disable security guard. Example: /guard?token=yourToken&enable=true

/imgur

Upload image to Imgur. Example: /imgur?link=example.com/image.jpg

/lookup

Lookup Facebook profile by link. Example: /lookup?fblink=fb.com/username

/pinterest

Fetch Pinterest pins. Example: /pinterest?search=dog&count=20

/remini

Enhance image with Remini. Example: /remini?url=example.com/image.jpg

/removebg

Remove background from image. Example: /removebg?url=example.com/image.jpg

/sms

Send an SMS. Example: /sms?number=1234567890

/translate

Translate text to a different language. Example: /translate?text=Hello&lang=fr
