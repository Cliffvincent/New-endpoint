Available Endpoints

DDOS

Spam a site via link. Example: /ddos/site/spam?link=https://:example.com

Emojimix

Mix two emojis. Example: /emojimix?emoji1=😂&emoji2=🤧

Fbshare

Facebook share bot. Example: /fbshare?link=example.com&token=yourToken&amount=10&speed=fast

Follow

Follow an account. Example: /follow?accountID=12345&accessToken=yourToken

Tempmail gen

Generate a temporary email.

Tempmail Inbox

Check temporary email messages. Example: /tempmail/message?email=temp@example.com

Guard

Enable or disable security guard. Example: /guard?token=your_Token&enable=true

Imgur

Upload image to Imgur. Example: /imgur?link=https://example.com/image.jpg

Lookup

Lookup Facebook profile by link. Example: /lookup?fblink=https://www.facebook.com/swordigo.swordslush

Pinterest

Fetch Pinterest pins. Example: /pinterest?search=dog&count=5

Remini

Enhance image with Remini. Example: /remini?url=https://example.com/image.jpg

Removebg

Remove background from image. Example: /removebg?url=https://example.com/image.jpg

Unsplash-HD

Unsplash search image hd example: unsplash cat count 1 to 5 images

Sms

Send an SMS. Example: /sms?number=095266434565

Translate

Translate text to a different language. Example: /translate?text=Hello&lang=fr
